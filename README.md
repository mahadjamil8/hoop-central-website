Hoop Central Website

This project is a basketball-themed website created for my Web Programming final. The website uses HTML5, CSS3, Bootstrap, JavaScript, and jQuery. It includes multiple pages about basketball players, teams, highlights, and rules.

Pages Included:
1. index.html - Home page
2. players.html - Top basketball players page
3. teams.html - Basketball teams page
4. highlights.html - Basketball highlights page
5. rules.html - Basketball rules and quiz page

Main Features:
- Responsive design using Bootstrap
- Custom CSS styling
- Basketball images, GIFs, and team logos
- Interactive team logo section using JavaScript/jQuery
- Highlights page using arrays, functions, localStorage, and jQuery class toggling
- Rules page with a multiple-choice quiz
- Built-in clock/date widget on the homepage
- Navigation bar linking all pages together

Folders:
- css: contains the style.css file
- js: contains the script.js file
- images: contains all images, GIFs, logos, and graphics used in the website

How to Open:
Open index.html in a web browser to start the website.
